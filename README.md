# Frequency-based Comprehensive Prompt Learning for Vision-Language Models 

## Running code
1. [Dassl-FCPrompt](Dassl-FCPrompt/) is the modified toolbox of [Dassl.pytorch](https://github.com/KaiyangZhou/Dassl.pytorch).
